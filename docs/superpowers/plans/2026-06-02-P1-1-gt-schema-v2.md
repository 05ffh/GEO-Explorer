# P1-1 GT Schema v2 实现计划 (修订版)

> **状态:** 专家审阅 ✅ → 补齐清单 ✅ → 修正计划 ✅ → 待用户确认

**Goal:** 将 Ground Truth 从扁平 JSON 升级为结构化 v2 schema，使每个品牌事实都能回答：字段是什么、候选值有哪些、主值是哪个、来自哪里、可信等级如何、谁确认过、能否支撑幻觉判定。

**Architecture:** JSONB v2 格式演进 + Field Registry + Pydantic 校验 + flat() 兼容层 + 字段级来源标注 + 迁移安全机制。

---

## 1. 现状

### 1.1 当前 GT 结构

```
GroundTruthVersion:
  ground_truth_json: JSONB  ← 扁平 {"field": value}
  source_urls: ARRAY[Text]  ← 全局来源，不关联字段
  reviewer: str             ← 全局审核人
  status: str               ← draft/active/archived
```

### 1.2 核心痛点

1. 无字段类型 → 校验靠 `GT_LIST_FIELDS` 硬编码
2. 每字段单值 → 无法表达多来源候选
3. 来源全局化 → 不知道哪个 URL 支撑哪个字段
4. 无法追溯 → 报告无法解释"判断依据是什么"

### 2.2 Source Tier 定义

| Tier | 含义 | 可用于核验 | 示例 |
|------|------|:---:|------|
| **S** | 官方一手来源 | 强证据 | 官网、年报、官方公告、监管披露 |
| **A** | 权威第三方 | 强证据 | 政府/监管、权威数据库、行业协会 |
| **B** | 可信机构 | 可辅助 | 权威媒体、研究机构 |
| **C** | 普通网页 | 弱辅助 | 百科、聚合页 |
| **D** | 未验证/低可信 | 不用于核验 | 自媒体、论坛、截图 |

规则: P0/P1 幻觉判定至少需 S/A/B 来源; D 级不得作为 contradicted 唯一证据。

### 2.3 Primary 规则

| field_type | primary 规则 |
|---|---|
| string | 必须且仅一个 primary |
| number | 必须且仅一个 primary |
| object | 必须且仅一个 primary |
| list | 允许多个 primary (每个 item 独立) |
| aliases | 允许多个 primary |

校验: string/number/object 多 primary → invalid; 无 primary → 自动选第一个 + warning。

### 2.5 Pydantic 模型

```python
# src/schemas/gt_v2.py
class GtSource(BaseModel):
    tier: Literal["S", "A", "B", "C", "D"]
    source_type: Literal["official_site", "official_report", "regulatory",
        "industry_association", "media", "database", "manual", "other"] = "other"
    title: str = ""
    url: str = ""
    evidence_text: str = ""
    retrieved_at: str | None = None
    confirmed_by: str | None = None
    confirmed_at: str | None = None

class GtValue(BaseModel):
    value: str
    primary: bool = True
    sources: list[GtSource] = []

class GtField(BaseModel):
    field_type: Literal["string", "list", "number", "object"]
    values: list[GtValue] = []
    status: Literal["draft", "reviewed", "verified", "deprecated"] = "draft"

class GtMeta(BaseModel):
    schema_version: Literal["gt_meta_v1"] = "gt_meta_v1"
    last_reviewed_by: str | None = None
    last_reviewed_at: str | None = None
    coverage_score: float = 0.0
    total_fields: int = 0
    completed_fields: int = 0
    required_fields: int = 0
    completed_required_fields: int = 0
    missing_required_fields: list[str] = []

class GroundTruthV2(BaseModel):
    schema_version: Literal["gt_v2"]
    fields: dict[str, GtField]
    meta: GtMeta = GtMeta()
```

### 2.6 兼容层

```python
def detect_gt_schema_version(gt_json: dict, db_col: str | None = None) -> str:
    if db_col == "gt_v2": return "v2"
    if isinstance(gt_json, dict) and gt_json.get("schema_version") == "gt_v2":
        return "v2"
    return "v1"

def gt_v2_to_flat(gt: GroundTruthV2) -> dict:
    result = {}
    for field_name, field in gt.fields.items():
        if field.field_type == "list":
            result[field_name] = [v.value for v in field.values if v.primary]
        else:
            primary = next((v for v in field.values if v.primary), None)
            if primary: result[field_name] = primary.value
            elif field.values: result[field_name] = field.values[0].value
    return result

def compute_coverage_score(fields: dict, registry: dict) -> float:
    required = {k for k, d in registry.items() if d.required}
    if not required: return 0.0
    completed = sum(1 for k in required if k in fields and fields[k].values)
    return round(completed / len(required), 4)
```

### 2.7 GroundTruthVersion 模型增强

```python
# src/models/ground_truth.py 新增
gt_schema_version: Mapped[str | None] = mapped_column(String(20), nullable=True)

def get_flat_json(self) -> dict:
    v = detect_gt_schema_version(self.ground_truth_json, self.gt_schema_version)
    if v == "v1": return self.ground_truth_json
    return gt_v2_to_flat(GroundTruthV2.model_validate(self.ground_truth_json))

def get_field_sources(self, field_name: str) -> list[dict]:
    v = detect_gt_schema_version(self.ground_truth_json, self.gt_schema_version)
    if v == "v1": return [{"tier": "C", "url": u} for u in (self.source_urls or [])]
    field = self.ground_truth_json.get("fields", {}).get(field_name, {})
    sources = []
    for val in (field.get("values") or []):
        for src in (val.get("sources") or []):
            sources.append(src)
    return sources

def get_best_source_for_field(self, field_name: str, min_tier: str = "B") -> dict | None:
    """Return highest-tier source >= min_tier for hallucination evidence."""
    tier_rank = {"S": 5, "A": 4, "B": 3, "C": 2, "D": 1}
    min_rank = tier_rank.get(min_tier, 3)
    best = None
    best_rank = 0
    for src in self.get_field_sources(field_name):
        r = tier_rank.get(src.get("tier", "C"), 2)
        if r >= min_rank and r > best_rank:
            best = src
            best_rank = r
    return best
```

## 6. 风险与缓解

| 风险 | 概率 | 缓解 |
|------|:--:|------|
| 消费者遗漏导致类型错误 | 中 | `get_flat_json()` 兜底 |
| 存量迁移丢信息 | 低 | backup + dry-run + flat 往返校验 |
| API 兼容破坏前端 | 中 | schema_version 字段，前端按版本渲染 |
| 性能退化 | 低 | flat() 在 Python 层，无 DB 负担 |
| Registry 字段不完整 | 中 | 未知字段 warning 不 block，可渐进添加 |
